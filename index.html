ChessMapper

I started this project at the same time as the website, and it is largely a way for me to showcase the advanced Java programming concepts I am developing through COMP308 (Java for Programmers) . It is also a program that I have a personal motivation to develop. I am currently studying Logical Chess by Irving Chernev , trying to get through a game on the board every couple of nights. I'm also playing on Chess.com when I find myself on my phone.

ChessMapper is a single-user chess study tool designed to simulate recorded games. The goal is to develop a GUI which you can feed chess notation or interact with using the mouse to play out the moves (of both white and black) of a recorded game. The main application provides the logic for all the usual restrictions of a chess game, meaning that white/black may only move on their respective turns and only legitimate moves are allowed to be made. The user will be able to rewind and redo turns. Using object persistence, the program will allow the user to save games that have been played out in the program to be loaded later from the main program.

ChessMapper is a program I am very excited to publish! I expect to have it finished by the end of April, and will begin to put my free-time effort into it after the final version of this website is published.

Drone Flight Control System

This is a project I have had on the back-burner for some time. I am excited however, to begin applying some of the Operating Systems concepts I have learned, and some of the C programming skills I have developed in COMP314 (Operating Systems) to the flight controller and radio control module of my hobby quad-copter build.

I intend to utilize public 3D-printers and open-source STL files to create the plastic body and props of the quad-copter. I already have most of the microelectronics required, and I've been practicing my finer soldering skills on various insignificant microelectronic projects (i.e., my jack-o-lantern for 2025 was a 3-pin RGB LED connected to a teensy microcontroller which had a synchronized fade of the three colour values to produce an orange glow).
